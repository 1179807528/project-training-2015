# 2015级项目实训成果展示 

## 《OD减肥》 - Java与大数据分析

### 项目简介

减肥是一款拥有大量减肥课程，并且可以根据兴趣爱好推荐课程的致力于改善大家的健身减肥方式的网站。

由于物质条件的改善和饮食结构的不尽合理，肥胖症发病率急剧上升，因肥胖及其并发症而致死的现象已成为日趋严重的社会问题。人们的健康意识逐渐增强，开始追求健康生活。我们就是针对这一社会特点而开发的一款可以让用户找到更适合自己减肥健身方式的网站。
