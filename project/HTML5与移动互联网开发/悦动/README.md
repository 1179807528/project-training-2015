### 项目简介

生活水平在提高，学生坐在教室学习，白领坐在办公室办公，很多人一天下来也没有多少运动量，对身体健康有明显的影响。

**悦动**为喜爱运动者提供了一个信息交流的平台。你可以在这里发布活动，让别的小伙伴参加。也可以和别的小伙伴约一场酣畅淋漓的运动。
