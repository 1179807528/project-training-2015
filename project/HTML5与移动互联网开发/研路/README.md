# 2015级项目实训成果展示 

##  《研路》  - HTML5与移动互联网开发

**Github：**[https://github.com/yanlu666/Yanlu](https://github.com/yanlu666/Yanlu)

### 项目介绍

**研路**是一款为考研大学生和研究生设计的一款学习、交流的APP。研路漫漫，我们伴你成长！  
这款APP主要是为了给在校大学生提供一个平台，让大学生在考研路上可以及时了解考研信息，考研资料。所以我们提供了好友通信、查找资料、设置任务、发帖、查看研讯的这五大功能。
