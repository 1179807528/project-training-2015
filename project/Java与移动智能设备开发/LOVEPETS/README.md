# 2015级项目实训成果展示 

## 《LOVE PETS》 - Java与移动智能设备开发

### 项目简介

LOVE PETS是一款针对养宠物的用户推出的宠物类APP。在这里你可以了解关于宠物最新最火热的资讯、可以在广场上分享你和爱宠的点滴、可以查阅各类宠物的养护方法、与众多爱宠人士进行在线交流沟通。
