# 2015级项目实训成果展示 

## 《掌上图书馆》 - Java与移动智能设备开发

### 项目简介

一款为在校学生和教师提供所在学校图书馆的馆藏信息，和个人的图书借阅情况的APP。旨在为各大高校学生开创图书馆书籍借阅平台，以实现节约时间以及高效阅览书籍，提升大学生文化素养。
