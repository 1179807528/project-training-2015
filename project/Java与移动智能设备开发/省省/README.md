# 2015级项目实训成果展示 

## 《省省》 - Java与移动智能设备开发

### 项目简介

省省APP是一款面向与大众消费者，给消费者提供快捷优惠信息整合的平台。当前市面上大部分的APP只是给大家提供了优惠后的价钱，或者需要在该平台中购买，对线下多种类的商品整合的也不太好，所以省省应运而生，省省平台将为您提供线下多种类商品的优惠信息，让您在出门前就做好决定去哪，省时省力！
