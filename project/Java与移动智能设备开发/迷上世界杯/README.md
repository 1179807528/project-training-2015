# 2015级项目实训成果展示 

## 《迷上世界杯》 - Java与移动智能设备开发

### 项目简介

《迷上世界杯》是一款面向足球爱好者的足球新闻类APP，为足球爱好者提供一手足球比赛资料，赛场奇闻异事，足球明星的八卦新闻，比赛的最新赛程，比赛结果，各个球队球员信息。
