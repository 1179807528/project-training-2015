## 《莫方行》 - Java与移动智能设备开发

### 项目简介

本产品坚持以为大学生提供最优质、最全面的服务为核心，专一服务于大学生为准则，主要是以一站式服务为理念，只做大学生旅游之一板块，更加具有针对性，此APP基涵盖了你所在城市周边的所有景点，用户可以选择多种方式出游，您可以自己自驾游，同时也可发布旅行消息招募志同道合的小伙伴一同前往，既游玩又交友。可以根据自己的需要来进行选择。从而方便人们的出行。
