## 《篮球吧》 - Java与移动智能设备开发

### 项目简介

篮球吧APP提供线上约球、线下打球、实时篮球咨询、NBA赛事直播、CBA赛事直播、著名球星动态、交流社区、篮球设备零售等功能，并依据APP的特点与线下高校合作。用户可在APP开辟自己的空间，通过个人主页分享自己的赛事直播赚取观看用户的赏金，分享打球心得，结交大量热爱篮球运动的朋友；各大高校也可利用此网站开展基于校园篮球赛事等活动，以达到大学生活跃开展篮球健身的目的。给你一个NBA梦。
